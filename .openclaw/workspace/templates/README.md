# 模板库

可复用的项目模板和文档模板。

## 📋 可用模板

### 项目模板
- `project-template/` - 标准项目结构
  - `README.md` - 项目说明模板
  - `.gitignore` - Git 忽略规则

### 文档模板
- 待添加

## ➕ 添加新模板

1. 在 `templates/` 下创建模板目录
2. 添加模板文件和说明
3. 在此文件更新列表

## 💡 使用建议

- 新项目从 `project-template/` 复制开始
- 常用文档格式保存为模板
- 定期更新模板反映最佳实践
