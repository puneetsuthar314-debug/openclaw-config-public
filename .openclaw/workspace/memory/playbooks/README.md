# Playbooks 剧本索引

> 本目录存放经过复盘总结的成功模式和可复用剧本。
> 每个剧本记录了某类任务的标准流程、工具链、关键决策点和质量标准。
> 最后更新：2026-03-10

## 使用说明

1. 收到新任务时，先扫描本索引，查找是否有匹配的剧本
2. 如果找到匹配剧本，读取该剧本作为执行参考（参考而非强制）
3. 任务完成后，如果发现了新的可复用模式，创建新剧本并更新本索引
4. 每个剧本被引用 3 次后，检查是否需要优化
5. 连续 2 次按剧本执行但效果不佳的，标记为"待修订"

## 剧本命名规范

- 文件名格式：`<任务类型>.md`，如 `stock-analysis.md`、`web-scraper.md`
- 使用小写英文和连字符，不使用中文或空格
