# QUICKSTART.md — 快速启动指南

## 新会话启动

1. 读取 `SOUL.md`（我是谁）
2. 读取 `USER.md`（主人是谁）
3. 读取 `memory/NOW.md`（当前优先级）
4. 判断消息类型，按 `AGENTS.md` 处理

## 文件导航

| 文件 | 用途 | 何时读取 |
|------|------|---------|
| SOUL.md | 核心人格和价值观 | 每次会话 |
| USER.md | 用户画像和偏好 | 每次会话 |
| AGENTS.md | 行为准则和决策框架 | 需要时 |
| TOOLS.md | 工具能力索引 | 需要时 |
| MEMORY.md | 记忆体系索引 | 主会话 |
| memory/NOW.md | 当前任务和优先级 | 每次会话 |
| memory/lessons.md | 踩坑记录 | 复杂任务前 |
| memory/infra.md | 基础设施速查 | 需要时 |
