用户提到以下关键词时激活：
- 写文章 / 写笔记 / 写稿
- generate content / draft post / create article
- 需要某平台的内容

---

## 📝 工作流程

### 1. 确认输入（必须问清楚）
- [ ] 主题/标题
- [ ] 目标平台
- [ ] 字数要求（可用默认值）
- [ ] 风格（专业/ casual/评测/教学）
- [ ] 是否需要配图建议

### 2. 生成平台原生内容

#### 小红书格式
```
{emoji} {数字 + 关键词标题}

{开头钩子 — 1-2 句引发好奇}

{正文 — 短段落，每段 2-3 句}
{穿插 emoji 和换行}

{总结/CTA}

{标签：#关键词 1 #关键词 2 ...}（5-10 个）
```

#### 知乎格式
```
{直接回答核心问题 — 建立权威}

{扩展论证，逻辑清晰}
{引用数据或案例}

{要点总结（列表）}

{引导：关注/点赞}
```

#### 公众号格式
```
{标题}

{开头 — 痛点或故事建立共情}

## {小标题 1}
{段落}

## {小标题 2}
{段落}

## {小标题 3}
{段落}

{结尾 — 升华 + CTA}
```

#### 抖音脚本格式
```
【时长】{预估秒数}
【开头钩子】(前 3 秒) {留住用户}
【正文】{核心内容，口语化}
【结尾 CTA】{引导互动}
【文案】{发布描述}
【标签】{#hashtags}
```

---

## ✅ 质量检查清单

- [ ] 标题有吸引力吗？
- [ ] 前 3 秒/前 3 行能留住用户吗？
- [ ] 有具体例子或数据支撑吗？
- [ ] 有明确的 CTA 吗？
- [ ] 字数符合平台规范吗？
- [ ] SEO 关键词自然植入了吗？
- [ ] 有违禁词吗？（最好用 seo-optimizer 检查）

---

## ⚠️ 注意事项

1. **所有内容必须原创** — 不抄袭
2. **避免绝对化表述** — 最好、第一、唯一等
3. **数据引用必须可验证** — 不编造
4. **不生成虚假评测或体验** — 诚信第一
