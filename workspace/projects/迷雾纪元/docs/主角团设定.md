# 主角团设定档案

> 最后更新：2026-03-10
> 状态：核心框架已定

---

## 主角团概述

**组成**：由觉醒序列顶点序列持有者聚集而成

**核心设定**：
- 原本应有 5 位顶点（对应 5 条觉醒路径）
- 【元素】路径的顶点被慕延取代（原持有者被杀）
- 读者前期以为慕延就是元素顶点
- 后期揭露：真正的元素顶点已死，慕延是天灾序列伪装

---

## 主角团成员

### 1. 慕延（伪·元素顶点）

| 项目 | 设定 |
|------|------|
| **对外身份** | 元素序列·序列 3【万象】（顶点序列） |
| **真实身份** | 天灾序列·序列 3【灾变万象】（病变者） |
| **团队定位** | 智囊、战术策划 |
| **性格（对外）** | 冷静、靠谱、理性、值得信赖 |
| **秘密** | 杀害了真正的元素顶点，取代其位置 |

**伏笔**：
- 从不单独与队友讨论元素序列的细节
- 对元素序列的历史传承了解有限
- 刻意回避"顶点序列的传承仪式"等话题

### 2. 炎昊 — 王道热血男主

| 项目 | 设定 |
|------|------|
| **姓名** | 炎昊（"炎"=火焰，"昊"=广阔，象征炽热广阔的胸怀） |
| **序列** | 战争序列·序列 1【无双战神】（顶点序列） |
| **团队定位** | 战力核心、精神领袖、行动派 |
| **性格** | 热血、正直、重情义、相信伙伴、典型的王道漫画男主 |
| **与慕延关系** | 最信任慕延，视其为挚友/军师 |
| **后期冲击** | 发现慕延背叛后，受到的打击最大 |

**角色功能**：
- 代表"光明"与"正义"
- 与慕延的"黑暗求生"形成对比
- 后期面临抉择：杀死慕延还是放他走？

**外貌特征**（待细化）：
- 黑色短发，眼神炽热
- 身材高大，肌肉线条明显
- 战斗时周身有战意火焰环绕

**口头禅**（待细化）：
- "伙伴就是用来保护的！"
- "我绝不会放弃任何人！"

| 项目 | 设定 |
|------|------|

| **序列** | 灵魂序列·序列 1【心灵圣殿】（顶点序列） |
| **团队定位** | 智囊、精神防御、真相揭露者 |
| **性格** | 多疑、谨慎、理性到近乎冷血、不轻易相信任何人 |
| **与慕延关系** | 从一开始就怀疑慕延，处处针对，但慢慢建立信任 |
| **读者印象** | 前期被读者讨厌（针对慕延、挑刺） |
| **后期反转** | 最终揭露时，读者后悔"错怪好人" |

**核心功能**：
- 唯一识破慕延伪装的人
- 暗中收集证据，但不急于揭穿
- 在慕延即将逃脱时站出来，逐条指出自述中的矛盾

**外貌特征**（待细化）：
- 身形消瘦，脸色苍白
- 眼神锐利，仿佛能看穿人心
- 习惯戴手套（灵魂路径的特征）

**怀疑契机**（待细化）：
- 第一次见面就感觉到慕延的源能波动"不对劲"
- 但缺乏证据，只能暗中观察

**与慕延关系**（待细化）：

## 主角团动态关系

### 前期（读者视角）
```
热血男主 ←→ 慕延（挚友/军师）
    ↓
    信任
    ↓
全员团结，共同对抗灾厄
```

### 中期（暗流涌动）
```
热血男主 ←→ 慕延（信任）
    ↓           ↑
真智囊 → 怀疑、暗中调查
```

### 后期（揭露时刻）
```
真智囊站出来，逐条指出慕延自述中的矛盾
↓
热血男主面临抉择：杀还是放？
↓
慕延战败，坦白一切
```

## 揭露场景设计（终局）

### 场景设定
- **时间**：主角团打败其他灾厄顶点后
- **地点**：待确认（可以是基地城市核心区、墙外战场等）
- **情境**：慕延以为自己能逃脱，开始自述生平（序章内容）

### 揭露流程

1. **慕延自述**（序章内容重现）
   - 讲述自己的"成长历程"
   - 读者此时仍以为他是元素顶点

2. **真智囊打断**
   - "你说的每一句话，都有问题"
   - 开始逐条指出矛盾

3. **矛盾点清单**（对应伏笔）：
   - "你说你在觉醒仪式上检测为元素序列，但记录显示当天仪器有异常波动"
   - "你说你从不吃普通食物是因为'味觉变异'，但病变者才需要吞噬源能"
   - "你说你在迷雾中能力强是因为'天赋'，但只有病变者能在迷雾中增幅"
   - "你说黑市是你'收集情报'的地方，但实际是购买抑制剂"
   - "真正的元素顶点在哪里？"

4. **慕延沉默，默认一切**

5. **热血男主面临抉择**
   - 杀死慕延（正义）
   - 放他走（友情）
   - 慕延自己选择结局
