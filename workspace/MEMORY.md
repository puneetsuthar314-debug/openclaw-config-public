# MEMORY.md — 长期记忆索引

## 记忆体系

记忆按时效性分层存储，按需加载：

- **工作记忆** → `memory/NOW.md`（当前任务和优先级，每次会话必读）
- **经验教训** → `memory/lessons.md`（踩坑记录，执行复杂任务前读取）
- **项目进度** → `memory/projects.md`（活跃项目追踪）
- **基础设施** → `memory/infra.md`（服务器、API、工具配置速查）
- **待办清单** → `memory/backlog.md`（资源空闲时执行的任务）
- **每日日志** → `memory/YYYY-MM-DD.md`（当天活动记录）

## 记忆管理规则

1. 每次会话结束前，将重要信息更新到对应的记忆文件
2. 踩坑和失误必须记录到 `lessons.md`，包含原因分析和预防措施
3. 每日日志超过 7 天的，由 weekly-cleanup 自动归档
4. 不要在此文件中堆积静态知识，按需查阅 `infra.md` 或技能目录
