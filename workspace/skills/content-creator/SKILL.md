# Content Creator

This skill is your all-in-one assistant for creating high-quality content. It combines idea generation, research, platform-specific drafting, and iterative refinement into a single, powerful workflow.

## When to Use This Skill

- Brainstorming content ideas for any niche or topic.
- Writing blog posts, articles, newsletters, or case studies.
- Generating publish-ready posts for Chinese social media (小红书, 知乎, 公众号, 抖音).
- Researching topics and adding proper citations.
- Improving the structure, hook, and flow of your writing.
- Getting real-time, section-by-section feedback on your draft.

## Core Workflow

The content creation process is divided into four phases. Follow them in order for the best results.

### Phase 1: Ideation & Strategy

Before writing, define your plan. If the user doesn't provide these, ask for them.

1.  **Brainstorm Ideas**: If the user needs inspiration, generate ideas first.
    *   `"Give me 10 blog post ideas for [niche/industry]."`
    *   `"Create a week's worth of social media content about [topic]."`

2.  **Define the Strategy**:
    *   **Topic & Angle**: What is the core message?
    *   **Target Audience**: Who are you writing for?
    *   **Platform**: Where will this be published? (e.g., Blog, 小红书, 知乎)
    *   **Goal**: What should this content achieve? (educate, persuade, entertain)
    *   **Style**: What is the desired tone? (formal, conversational, technical)

### Phase 2: Outlining & Research

Create a strong structure before drafting the content.

1.  **Collaborate on an Outline**: Work with the user to build a logical structure. Use the template from `content-research-writer` for detailed outlines.

2.  **Conduct Research**: As you identify gaps in the outline, use search tools to find credible information.
    *   `"Research [specific topic] and add citations to my outline."`
    *   Extract key facts, quotes, and data.
    *   Add citations in the user's preferred format (e.g., inline, numbered, footnote).

### Phase 3: Platform-Specific Drafting

Write the content using formats optimized for the target platform. Always check for and avoid banned or sensitive words, especially for social media platforms.

#### General Article/Blog Post

Follow the structure from the outline. Focus on a strong hook, clear arguments supported by evidence, and a compelling conclusion.

#### 小红书 (XiaoHongShu)

```
{emoji} {Title with numbers + keywords}

{Opening hook — 1-2 sentences to spark curiosity}

{Body — short paragraphs, 2-3 sentences each, using emojis and line breaks}

{Summary / Call to Action}

{Tags: #keyword1 #keyword2 ...} (5-10 relevant tags)
```

#### 知乎 (Zhihu)

```
{Direct answer to the core question to establish authority}

{Expanded argument with a logical structure, citing data or case studies.}

{Use bullet points for a scannable summary.}

{Nudge the reader to follow or upvote.}
```

#### 公众号 (WeChat Official Account)

```
{Compelling Title}

{Introduction — build empathy with a pain point or story.}

## {Subtitle 1}
{Detailed paragraph}

## {Subtitle 2}
{Detailed paragraph}

{Closing — summarize, elevate the message, and include a clear Call to Action.}
```

#### 抖音 (Douyin) Script

```
【Duration】{estimated seconds}
【Opening Hook】(first 3 seconds) {A strong visual or statement to stop the scroll}
【Body】{Core content delivered in a conversational, engaging tone}
【Closing CTA】{Drive engagement: like, comment, follow}
【Caption】{Post description with keywords}
【Tags】{#relevant_hashtags}
```

### Phase 4: Review & Refinement

Act as a writing partner to polish the draft.

1.  **Improve the Hook**: Review the introduction and suggest more compelling alternatives (e.g., a bold statement, a personal story, surprising data).

2.  **Provide Section-by-Section Feedback**: As the user writes, review each section for clarity, flow, evidence, and style. Use the detailed feedback template from `content-research-writer`.

3.  **Conduct a Final Review**: Once the draft is complete, perform a full review covering structure, content quality, technical accuracy (grammar, citations), and readability.

4.  **Use the Pre-Publish Checklist**:
    *   [ ] Is the title compelling and optimized?
    *   [ ] Does the hook grab the reader's attention?
    *   [ ] Are all claims supported by credible sources?
    *   [ ] Is there a clear Call to Action?
    *   [ ] Is the formatting native to the target platform?
    *   [ ] Has it been proofread for typos and grammatical errors?
