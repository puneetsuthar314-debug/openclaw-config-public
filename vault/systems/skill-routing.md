# 技能路由指南

当多个技能看起来都能处理用户请求时，按以下规则选择：

## 浏览器操作
- 访问外部网站、填表单、爬数据 → `agent-browser`
- 需要高质量截图、稳定标签控制 → `agent-browser`
- 测试本地运行的 Web 应用 → `webapp-testing`

## 数据分析
- 通用结构化数据（CSV/Excel/JSON）→ `data-analysis-unified`
- 科学数据、统计建模、出版级图表 → `scientific-analysis`
- 股票/公司/金融市场 → `stock-analysis`
- 网站流量/域名分析 → `similarweb-analytics`
- Excel 文件读写编辑 → `excel`

## 内容处理
- 从 URL 抓取/下载内容 → `content-fetcher`
- 创作文章/博客/社媒内容 → `content-creator`
- 从 URL 学习并转化为行动计划 → `learn-and-ship`
- 文档/长文本快速摘要 → `summarize`

## 搜索调研
- 快速搜索/深度调研/研究报告 → `unified-search`
- 行业分析/市场研究（50+页报告）→ `market-research-reports`
- 查找/推荐新技能 → `skill-finder`

## 安全相关
- 工作区文件完整性检查 → `soul-guardian`
- 全面安全审计/漏洞扫描 → `security-suite`
